[ Stealth Logger ]

> Description

This is a stealth IP logger written in PHP. It collects the visitor's IP address, User-Agent, and Referrer, logs it into a file, and silently redirects the visitor to a safe destination (e.g., Google).

> Features

  • Silent redirection after logging
  • Lightweight and server-friendly
  • Ideal for tracking phishing/shell interactions

> Code Snapshot

<?php $logFile = "ips.txt"; $ip = $_SERVER['REMOTE_ADDR']; $userAgent = $_SERVER['HTTP_USER_AGENT']; $referrer = $_SERVER['HTTP_REFERER'] ?? "N/A"; $timestamp = date("Y-m-d H:i:s"); $logEntry = "$timestamp | IP: $ip | UA: $userAgent | Referrer: $referrer\n"; file_put_contents($logFile, $logEntry, FILE_APPEND); header("Location: https://www.google.com"); exit; ?>
